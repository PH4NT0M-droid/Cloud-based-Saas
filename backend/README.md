# Cloud-Based SaaS OTA Channel Manager

## Overview
A full-stack OTA channel manager for hotel/property operations with backend APIs, frontend dashboard, analytics, OTA sync, inventory, bookings, and deployment setup.

## Features
- Authentication and role-based access control
- Property and room management
- Inventory and rate management
- OTA sync adapters
- Booking lifecycle management
- Analytics dashboard
- Production deployment configuration

## Tech Stack
- Backend: Node.js, Express, Prisma, PostgreSQL
- Frontend: React, Vite, Redux Toolkit, Tailwind CSS
- Tooling: Jest, Vitest, Docker, Nginx

## Setup
1. Install dependencies in `backend` and `frontend`
2. Configure environment variables from `.env.example`
3. Run database migrations
4. Start backend and frontend
